# Automobile Exploratory Data Analysis
## Description
Buying a car can be complicated with all of the options available. This exploratory data analysis and [report](https://github.com/kaylynvanas/automobile_eda/blob/master/EDA.docx) identifies what the average consumer can look out for when buying an automobile.

The analysis focuses on:
- the effects of body style on price;
- the effects of engine size on price;
- the makes of the cheapest automobiles; and
- the makes of the most expensive automobiles.

## Run
To run:
- Download the [Jupyter Notebook](https://github.com/kaylynvanas/automobile_eda/blob/master/automobile.ipynb) and [Data Set](https://github.com/kaylynvanas/automobile_eda/blob/master/automobile.txt) to a folder;
- Open your terminal or command prompt and change directory to the folder using `cd`;
- Enter `jupyter notebook`;
- Once Jupyter Notebook is open, click Cell -> Run All in the menu.
